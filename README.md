# Simple HTML Tools

A collection of small, self-contained browser tools. Each is a static HTML file you can just double-click to open — no server or build step needed. Only `qr-reader.html` needs an internet connection.

### qr-reader.html
Read QR codes from an image, PDF, or pasted screenshot (Ctrl/Cmd+V), via drag-and-drop or file picker.

### vector-visualizer.html
Add and edit 3D vectors by their x/y/z components and view them in an interactive 3D plot.

### video-sync-mapper.html
Load two videos side by side and find the linear time mapping that lines them up, either by hand or via two-point calibration.

### video-annotator/
Annotate single frames or frame ranges in a video with custom, keyboard-shortcut-driven event labels, then export/import them as a text file. Open `video-annotator/index.html` (keep `app.js` alongside it).

### neuron-color-adjuster.html
Drop in an image and tune hue, saturation, and brightness with sliders, comparing the original and adjusted versions side by side.
